# 🔐 Authentification Paroisse - Guide Complet

Bienvenue! Voici le guide complet du système d'authentification sécurisé de **Paroisse**.

---

## 📱 1. **Inscription (Signup)**

### Étapes :

1. Cliquez sur **"S'inscrire"**
2. Remplissez vos informations :
   - Email
   - Mot de passe (au moins 8 caractères)
   - Nom & Prénom
   - Téléphone
   - Avatar (photo de profil) - _optionnel_
3. Cliquez **"S'inscrire"**

### ✅ Ce qui se passe :

- Votre compte est créé de manière sécurisée
- Un **email de confirmation** vous est envoyé
- Votre profil est préparé pour la première connexion

---

## 📧 2. **Confirmation d'Email**

### Étapes :

1. Vérifiez votre **boîte mail** (ou spam)
2. Ouvrez l'email de mis dans le site **Paroisse**
3. Cliquez sur le lien **"Confirmer mon email"**
4. Une page de confirmation s'affiche
5. Vous êtes automatiquement redirigé

### ✅ Ce qui se passe :

- Votre email est vérifié
- Votre profil est finalisé (avec avatar si fourni)
- Vous pouvez maintenant **vous connecter**

---

## 🔑 3. **Connexion (Login)**

### Étapes :

1. Cliquez sur **"Se connecter"**
2. Entrez votre **email** et **mot de passe**
3. Cliquez **"Se connecter"**

### ✅ Ce qui se passe :

- Vous êtes authentifié de manière sécurisée
- Votre profil se charge automatiquement
- Vous accédez à votre compte

---

## 🔒 4. **Mot de Passe Oublié**

### Étapes :

1. Sur la page de **connexion**, cliquez **"Mot de passe oublié?"**
2. Entrez votre **email**
3. Cliquez **"Envoyer le lien"**
4. Vérifiez votre **boîte mail**
5. Cliquez sur le lien **"Réinitialiser"**
6. Entrez un **nouveau mot de passe**
7. Cliquez **"Mettre à jour"**

### ✅ Ce qui se passe :

- Un lien sécurisé vous est envoyé
- Vous pouvez créer un **nouveau mot de passe**
- Vous pouvez vous connecter avec le nouveau mot de passe

---

## 🛡️ **Sécurité**

Nous utilisons :

- ✅ **Chiffrement SSL/TLS** pour protéger vos données
- ✅ **Authentification Supabase** (norme industrie)
- ✅ **Tokens sécurisés** pour votre session
- ✅ **RLS (Row Level Security)** pour isoler vos données
- ✅ **Validation des emails** pour éviter les abus

---

## ❓ **Questions Fréquentes**

**Q: Où va mon avatar?**  
R: Il est uploadé sur nos serveurs sécurisés et affiché dans votre profil.

**Q: Combien de temps pour recevoir l'email?**  
R: Généralement quelques secondes. Vérifiez votre **spam** si rien.

**Q: Je n'ai pas reçu l'email?**  
R: Rendez-vous sur la page de **connexion** et cliquez **"Renvoyer email"** ou utilisez **"Mot de passe oublié"**.

**Q: Puis-je changer mon mot de passe après connexion?**  
R: Oui! Rendez-vous dans **Paramètres → Sécurité**.

**Q: Mes données sont-elles sûres?**  
R: Oui! Nous utilisons les standards de sécurité les plus élevés avec des politiques RLS strictes.
